# neovim-conf
My Neovim Configuration with [LazyVim](https://github.com/LazyVim/starter) starter template for my backup/tracking

## Setup
Backup your current neovim configuration

```bash
mv ~/.config/nvim ~/.config/nvim.bak
```

Clone this repo in it's place

```bash
git clone https://github.com/jpisani0/neovim-conf ~/.config/nvim
```

Run `nvim` and wait for setup to complete
